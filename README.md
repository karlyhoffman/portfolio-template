# Portfolio Template One

## Project Deliverables:
- A responsive portfolio website that showcases various projects.

## Technologies Used:
- HTML 5
- Sass/CSS
- Bootstrap 4
- JavaScript/JQuery
- GreenSock Animation Platform (GSAP)
- ScrollMagic
- Gulp

## Getting Started:
1. Download or clone repository
2. Run `npm install`
3. Run `gulp`
